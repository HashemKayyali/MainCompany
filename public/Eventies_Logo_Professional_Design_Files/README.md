# Eventies Logo – Professional Design Files

This package contains cleaned exports of the Eventies logo prepared for website use and designer handoff.

## What was fixed
- Black wordmark cleaned to solid pure black: `#000000`.
- White wordmark cleaned to solid pure white: `#FFFFFF`.
- Transparent backgrounds prepared for both light and dark use.
- Rough/jagged/dark edge artifacts were removed as much as possible from the raster exports.
- Web-ready PNG, WebP, SVG wrappers, favicon, Apple, Android, and PDF preview files were generated.

## Recommended files
- Light backgrounds: `01_master_png/eventies-logo-master-black-wordmark-transparent.png`
- Dark backgrounds: `01_master_png/eventies-logo-master-white-wordmark-transparent.png`
- Website optimized: use files inside `02_web_png` or `03_webp`
- Favicon/app icon: use files inside `05_favicon_app_icons`
- Preview: `00_preview/eventies-logo-clean-package-preview.png`

## Designer note
The SVG files in this package are web-safe SVG wrappers with embedded clean PNG artwork, not fully editable native vector paths.
For a 100% editable Illustrator/Figma source, the designer should rebuild/outline the final wordmark using a licensed Zodiak font file and match the included PNG master.
No font files are included in this package.
